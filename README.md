# SteedOS (with backend)

## Run it

1. Install Node.js 20+.
2. Open this folder in Terminal.
3. Run:

```bash
npm start
```

4. Open: `http://localhost:8787`

## What this now does

- Serves your app from a local backend server
- Saves app state to `data/steedos-db.json`
- Saves tomorrow plans in the same backend file
- Keeps localStorage fallback so it still works if backend is unavailable
